⚙️ Configurar sincronización con Google

Solo necesitas hacer esto una vez. Después, tus monas se sincronizan solas entre PC y celular con tu cuenta de Google.

1Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google.
2Clic en "Crear un proyecto" → ponle el nombre que quieras (ej: "mis-monas") → continuar hasta crear.
3En el menú izquierdo → Compilación → Realtime Database → Crear base de datos → Modo de prueba → Listo.
4En el menú izquierdo → Compilación → Authentication → Comenzar → Google → Habilitar → Guardar.
5Clic en el ícono ⚙️ junto a "Descripción general del proyecto" → Configuración del proyecto → baja hasta "Tus apps" → ícono </> (Web) → registrar app → copia los valores de abajo.
